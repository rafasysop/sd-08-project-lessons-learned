Lições Aprendidas

  1. HTML é uma linguagem de marcação;
  2. CSS é uma linguagem de estilo;
  3. Nenhuma das duas são linguagens de programação;
  4. Elementos HTML semânticos devem ser priorizados;
  5. Existem dezena de elementos HTML e ninguém decora todos;
  6. Existem centenas de atributos CSS e ninguém decora todos mesmo;
  7. Dá pra fazer qualquer layout possível com CSS o suficiente;
  8. Todo bom frontender domina HTML e CSS;
  9. Dominar esse par maravilhoso não é decorar tudo;
  10. Dominar esse par maaaaaravilhoso é saber extrair o potencial que ele oferece;
