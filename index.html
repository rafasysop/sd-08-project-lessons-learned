Sobre conhecimento

Um conhecimento nunca é concreto, sempre se tem algo mais a aprender. Os exemplos contidos nessa página são resumos do que experienciei esse mês. Não tome-os como conclusivos, pois cada tópico é bem mais profundo do que o conteúdo dessas pequenas amostras. Aqui você irá encontrar um insentivo, para utilizar materiais como este, como rota de estudos. Por fim, deixarei o link das documentações de cada tema em seus titulos.
Divirtam se.

GNU-Linux/ Bash

Sou entusiasta do mundo Unix, tento estar sempre por dentro de novas tecnologias. Goste de testar e aprender sobre variados sitemas Unix Like, como linux. Sou capaz de navegar e fazer diversas operações em um sistema unix sem necessitar de um ambiente grafico e tenho conhecimento para criar simples scripts para ajudar em tarefas rotineiras de Sistemas Unix.

Git

Uma ferramenta essencial para o trabalho do desenvolvedor é o Git. Uso essa ferramenta todos os dias e estou maravilhado com o sistema de versionamento.Esse me traz mais segurança e controle sobre meus projetos. Podendo, assim, checar alterações feitas nas versões anteriores.

GitHub

Assim como o Git, o GitHub é uma ferramenta que também auxilia no controle de versionamento de nossos codigos. Nele se pode criar um repósitorio remoto que será vinculado ao repositório local,podendo então fazer atualizações em ambos. Além do mais, o GitHub é uma plataforma que ajuda, e muito, em trabalhos em grupo e na troca de conhecimento. Nele é possivel deixar seu repósitorio público, fazendo com que outras pessoas possam baixar-lo e contribuir com o projeto. Tudo isso com sua autorização, é claro. O GitHub é acima de tudo uma ferramenta de troca de conhecimento!

HTML

O HTML é a Markdown que constroi a estrutura nos Desenvolvimentos Web. Atua como um documento que sera traduzido pelo browser, tento como retorno o conteudo contido nele. Nele é possivel colocar inumeros conteudos, como textos, links, imagens, entre outras midias. É possivel também colocar links para conteúdos internos e externos, assim podendo criar rotas para outros documentos, como o de estilização , o CSS.
