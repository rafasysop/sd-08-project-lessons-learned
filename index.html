Aprendizados

Meu aprendizado em desenvolvimento web foi iniciado quando decidi me inscrever no processo seletivo da Trybe. Primeiramente estudei todo o conteúdo disponibilizado pelo curso introdutório ao JavaScript para realizar o desafio prático. Depois que realizei o desafio, comecei a estudar sobre HTML e CSS pelo W3School. Com isso, adquiri os conhecimentos básicos sobre o desenvolvimento de páginas web.

O Conhecimento

Depois que foram iniciadas as aulas da Trybe, foi bastante interessante poder ajudar meus colegas da Turma 8 e ver que o conhecimento foi fixado.
